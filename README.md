# Frank's Browser-Based Tools

Various utilities and tools that run locally right in your browser.

## Dashboard

A centralized dashboard is now available! Open [index.html](index.html) in your browser to quickly access and search all tools in one place. The dashboard features:

- Search/filter bar for tools
- Mobile-friendly responsive design
- External CSS for easy customization
